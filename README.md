## McMMO
McMMO plugin for PocketMine-MP

## Download
https://poggit.pmmp.io/ci/AkmalFairuz/McMMO

## Usage
/mcmmo - to check your stats and leaderboard (UI)

/mcmmoadmin - to spawn leaderboard floating text
